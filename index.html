Public Access Submission System

PASS is an easy-to-use web application which supports and improves compliance to funding agencies' public and institution's open access policies. With PASS, an author accepted manuscript can be submitted to multiple repositories seamlessly at once. Researchers may be able to avoid paying Open Access publishing fees using PASS.
